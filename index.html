Desenvolvimento WEB

HTML é a base de uma página WEB, como esqueleto de um corpo. Ele define a estruturação de todos os elementos (títulos, parágrafos, divisorias, imagens, tabelas, etc.). Ele demarca muito bem uma página WEB, com início, meio e fim. Contudo, o HTML não se preocupa com beleza, tudo fica apenas um "preto e braco". E é ai que entra o CSS.

O CSS é o que dá estilo aos elementos, como a roupa e adereços que uma pessoa utiliza. O CSS define cores, tamanhos, efeitos visuais e, ao fim, deixa a página mais atrativa e muito, muito mais bonita. Ainda assim, somente com CSS a página continuará engessada, sem movimento, sem vida. Chegamos então ao Javascript.

O JavaScript é o que dá movimento e ação às páginas WEB, o cérebro do corpo. Ele é praticamente o espírito, que dá a vida ao corpo. Com o JavaScript pode-se criar interações, animações, gerando dinamismo à sua página e tudo ficará super atrativo para os usuários.
